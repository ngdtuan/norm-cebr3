# NORM CeBr₃ Mobile PWA

Gói GitHub Pages hoàn chỉnh cho tài khoản `ngdtuan`, repo `norm-cebr3`.

## File chính

- `index.html` — ứng dụng NORM CeBr₃, chạy trực tiếp trên trình duyệt.
- `manifest.json` — cấu hình cài đặt PWA trên Android.
- `sw.js` — service worker để chạy offline sau lần mở đầu tiên.
- `icons/icon-192.png`, `icons/icon-512.png` — biểu tượng ứng dụng.

## Triển khai trên GitHub Pages

1. Upload toàn bộ nội dung gói này lên repo `https://github.com/ngdtuan/norm-cebr3`.
2. Vào **Settings → Pages**.
3. Chọn **Deploy from a branch**.
4. Chọn branch **main**, folder **/(root)**, rồi **Save**.
5. Mở địa chỉ:

```text
https://ngdtuan.github.io/norm-cebr3/
```

## Cài trên Android

Mở link bằng Chrome Android → menu ⋮ → **Install app** hoặc **Add to Home screen**.

Ứng dụng sẽ chạy offline sau lần mở đầu tiên.
